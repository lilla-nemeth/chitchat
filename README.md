## Installing, running locally

Clone the repo

```
git clone https://github.com/lilla-nemeth/chitchat.git
```

### Backend

Go to the project root directory

```
cd chitchat
```

Install the dependencies

```
yarn install
```

Run the server

```
yarn run dev
```

Server is running on port 8080
</br></br>

### Frontend

Go to the client folder

```
cd frontend
```

Install the dependencies

```
yarn install
```

Start the frontend

```
yarn run dev
```
